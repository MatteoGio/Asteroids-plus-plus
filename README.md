# Asteroids-plus-plus

A simple implementation of the famous 1979 Atari game <b>[Asteroids](https://en.wikipedia.org/wiki/Asteroids_%28video_game%29)</b> realized in JavaScript using <b>[P5.js](https://p5js.org/)</b> libraries.
